# DigiMart

A web marketplace for buying and selling NFT's (Non-Fungible Token) with Ether as the primary cryptocurrency.

## Technologies
- MongoDB
- Mongoose.js
- Express.js
- React.js
- Node.js
- Bootstrap CSS
- Google Firebase
- Cloudinary
